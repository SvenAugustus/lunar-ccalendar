# lunar-ccalendar
Simple and lightweight QT Chinese lunar calendar.

 简单而轻量级的QT应用：中国日历，其中包含农历、节气、常见节假日。 

 
#### 环境(Enviorment)：
* QT version: QT4+  (Compiler: GCC recommended)

### 用法(Usage)：
* build & make (qtcreator recommended)
